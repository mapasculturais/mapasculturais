# O que é uma oportunidade apenas para divulgação?

Uma oportunidade apenas para divulgação é criada quando o objetivo é informar sobre ações, programas ou editais, mas sem que as inscrições sejam feitas pela plataforma.

Nesse modelo, a oportunidade serve apenas como um canal de comunicação e divulgação, onde os interessados podem consultar informações, mas o processo de candidatura ocorre externamente, por outros meios indicados na descrição da oportunidade.

> **Atenção:** As inscrições para esta oportunidade não serão feitas pela plataforma.
